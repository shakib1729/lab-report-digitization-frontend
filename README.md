# lab-report-digitization-frontend

The frontend part of project Lab Report Digitization created using ReactJS

Deployed on Heroku: https://lab-report.herokuapp.com/


Video Demo: https://youtu.be/KeDUEq5m3ns


To set up locally:

1) Clone
2) npm install
3) npm start
